# Install Markdown on Notepad ++

This is a guide of how to install Markdown language into Notepad++.
The steps are the following:

1. Download the Markdown language definition file:
	- Zenburn Theme : userDefineLang_markdown_zenburn.xml
	- Default Theme : userDefineLang_markdown.xml
		
2. In Notepad++ menu, click Language and select Define your language... .

3. In User Defined Language windows, click Import then open the xml file. 

4. Restart Notepad++.

5. Open and test with a Markdown file e.g. test.md
